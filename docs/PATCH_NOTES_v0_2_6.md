# Magpie TTS Studio v0.2.6

Documentation layout cleanup release.

## Changed

- Moved all versioned patch notes from the project root into `docs/`.
- Added `docs/PATCH_NOTES_v0_2_6.md` for this cleanup release.
- Updated the README project structure so the documentation folder is visible.
- Bumped app version to `0.2.6`.

## Why

The project root should stay focused on files users actually run or open first, such as `README.md`, `install_windows.bat`, `run_windows.bat`, and `app.py`. Version history is useful, but it belongs in the documentation area.
